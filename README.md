# DPDP-Compliance-Dashboard-UI
PrivacySetu – DPDP Compliance Dashboard UI Enterprise-grade, responsive frontend prototype for India’s DPDP Act, showcasing consent management, Data Principal request handling, compliance metrics, and trust-focused SaaS UI using pure HTML, CSS, JavaScript, and Chart.js.
